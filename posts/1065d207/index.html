    二合一电脑调教指南

    发表于 2023-03-11 更新于 2023-03-17 分类于 电脑调教

    文本介绍了在二合一电脑上实现高续航和高响应的解决方案。

    这个指南是我基于我当前的设备的调教经验写出来的，可以当做一个参考来看看。个人认为可以拓展成二合一电脑的调教方案。下方使用 平板 来代替 电脑 这个词，这样，你才可以知道我说的是二合一电脑（

    平板设置

    从所周知，如果一个设备运行流畅，那么要么这个设备的芯片的能效足够好，要么这个设备运行时所需要的运算量很小。可惜，这个是一个 windows 平板，同时还是一个基于 arm 处理器的二合一平板。可谓是集三者的缺点于一身了！不但这个芯片的能效不高，同时，运行 windows 还需要大量的计算量。

    因此，这个平板注定了要么牺牲续航来换取大量的计算量，从而来使得平板运行的流畅。要么牺牲系统运行时的流畅性，从而来使得平板获得长时间的续航。

    好，知道了这个基本的道理了以后，我来讲一下这个平板的适用的场景：需要长续航，高便捷性的工作。比如移动办公，视频网课等。不适用于需要大量计算的场景，比如：人工智能训练，游戏，工程建模等。当你有这个平板的时候，一定要有一个主力机（高性能计算机）。

    首先，我先讲一下这个设置涵盖的几个方面：

    • 基本的系统设置
    • 省电优化
    • 性能优化

    通过以下的设置，你将获得:

    • 长时间的续航（12 h+）
    • 流畅的体验

    注意：流畅指的是电脑在运行时不会或很少出现卡死的情况。并不代表你的平板可以进行大量计算。

    基本的系统设置

    在本块内容中，将介绍基本的 windows 设置。

    升级你的平板到 windows 11

    不用听别人说 windows 11 这不好用，那不好用，有多少 bug 等等。目前为止，我还没有遇到过 bug。而且，windows 11 添加了很多针对触屏的优化，所以是很有必要的。

    开启省电模式

    Windows 的省电模式可以让系统通过限制后台的程序的方法来帮助我们达到省电的目的。

    设置方法：

    这里建议将 自动开启节电模式 设置为 始终。平板在连接上电源以后，系统会自动关闭省电模式。

    尽可能的降低屏幕亮度

    如果将平板的屏幕调的太高，屏幕的功耗会显著增加，从而减少电脑的续航时间。

    设置方法：

    以上是两个 windows 的基本的设置方法。通过这些方法，已经可以让平板的续航得到一定的提升。

    不要使用动态壁纸

    动态壁纸会消耗大量的计算资源，会显著的减少电脑的续航时间。

    省电优化

    在本块内容中，将介绍 windows 的进阶设置方法以及通过一些软件来辅助我们达到更加省电的目的。

    设置电源计划

    电源计划是 Windows 系统中的一个功能，它可以让你根据不同的使用场景，调整电脑的性能和耗电量。通过电源计划，我们可以更加精准的控制电脑的耗电量。

    可以通过以下方法进入电源计划：

    进入页面以后，点击 更改高级电源设置 来编辑我们使用的电源计划。

    设置以下的内容：（可以根据自己的需要来调整）

    1. 睡眠 > 在此时间处睡眠 > 使用电池 > 3 分钟
    2. 睡眠 > 允许使用定时器 > 使用电池 > 禁用
    3. PCI Express > 链接状态电源管理 > 使用电池 > 最大电源节省量
    4. （重点）处理器电源管理 > 最小处理器状态 > 使用电池 > 1%
    5. （重点）处理器电源管理 > 最大处理器状态 > 使用电池 > 50%
    6. 显示 > 在此时间后关闭显示 > 使用电池 > 2 分钟
    7. 电池 > 低电量操作 > 使用电池 > 休眠

    这里解释一下可能的疑问：

    Windows 的在使用的过程中，为了可以即时响应用户的操作，因此会拉高功耗来加速响应，因此，可能会出现一个瞬时功耗飙升的情况。从而会使得 windows 的续航出现下降的情况。通过设置 最大处理器状态 ，可以限制处理器使用的最大功耗，从而来达到省电的目的。

    当然，一切设置都是有代价的，限制了功耗以后可能会出现响应不即时的情况（相比于没限制）。

    Windows 休眠是通过将你内存里的数据全部放到硬盘里，然后电脑关机，等你下一次使用的时候，再从硬盘中读取数据，然后快速的回到你之前的工作进度。

    休眠与关机的区别是：关机了以后，你再开机的时候需要重新打开程序，而休眠则可以不需要重新打开。

    下载 Energy Star X 软件

    能源之星 X 利用 Windows 11 的 EcoQos API（即“效率模式”）来限制后台应用的资源占用，从而提高电池续航和散热表现。它不会限制前台应用，以确保用户体验。
    摘自 Energy Start X

    下载地址：

    1. 微软商店
    2. github

    通过这个软件，可以显著的提高你的电脑的续航时间。

    下载 ParkControl 软件

    官网： ParkControl

    ParkControl 是一个用来控制 cpu 频率的软件，同时，它还可以自动的控制 cpu 启用的核心数量。可以通过降低 cpu 运行的核心数量来减少电脑的电源消耗。

    性能优化

    在本块内容中，将介绍用于优化电脑性能的方法。

    下载 Process lasso 软件

    官网：Process lasso

    Process Lasso 是用来动态平衡 cpu 负载的软件。它拥有比 windows 更加完善的资源平衡算法，并且会自动抑制高占用的程序，从而来优化用户对系统的使用体验。

    安装了这个软件以后，可以明显的感觉到系统出现卡死的情况大浮下降。

    该软件并不会占用很多电量（< 1% ），因此，对平板的续航没有任何影响。

    结尾

    最后，我再强调一下，以上所有的设置都是通过牺牲性能来换取更多的续航时间 。不存在通过我的设置以后，功耗低，性能强的情况。
