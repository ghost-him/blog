考研经验分享

发表于 2023-12-26 更新于 2024-04-18 分类于 经验分享

现在是刚刚考完试的第二天，写一下这个考研经验贴。

给各位一个参考，像数学，专业课什么的，要按自己的进度来。

最近蛮多人问我怎么复习的，所以我又在数学方面补充了一些内容，之前看过的同学可以再看看。

2024.4.18：受到学校邀请，上台演讲。现在将演讲稿同步到博客。

初试成绩

学习时间安排

这是我在冲刺阶段的日常安排，强化阶段，基础阶段的已经忘了。总体是这样的，各个时间在执行时期会有波动。误差不超过30min。

冲刺阶段是时间是12月以后，不休息。强化阶段，基础阶段如果学累了可以早回宿舍休息，打游戏等。休息时间定在晚上，白天非必要不休息。

时间 任务
2:00-9:30 睡觉
10:00-11:00 抄作文，背政治
12:00-(16:00/17:00) 写数学（写模拟卷，写错题等）
18:00-21:00 写408（写模拟卷，背错题本等）
23:00-24:00 背单词（白天等饭的时候也会背）
24:00-1:20 背肖四

备考期间一定要注意锻炼身体，我的频率是每隔 3 天跑一次步，一次大概在 1.2 公里，也就是跑 3 圈。在临考前半个月里，宿舍里的 3 个人都发烧了，就我还正常的，整个大四上学期都没有生病。

在整个备考期间一定要保证睡眠，至少要睡满7.5h(8h则更好)。睡眠充足对学习大有益处，推荐各位在考试结束以后阅读一下《我们为什么要睡觉？》这本书。

考研择校

考研预报名的时间在9月底，正好是在强化阶段快结束的时候，所以在报名前，可以自己去模拟一下。所以要把近3年的真题留下来，不要写，也不要看题目的内容。要对自己负责。

主要模拟的是数学，英语，专业课。如果专业课没来得及学完或者你选的学校今年的专业课改了，比如改成408，则可以只模拟数学和英语。

模拟的题目建议选择去年加前年的题目，考试的时间要严格限制在3小时。比如我参加的是24考研的，所以我就模拟23的真题和22的真题。我写一下我的扣分方法，可以给一个参考：数学如果一个大题有2问，总共12分，则第一问4分，第二问6分，结果算错，少加单位则扣2分，不给过程分。英语的翻译先算一下总共有几个句子，然后算一下一个句子多少分，句子翻译错了或者一个句子里出现2个及以上的错误单词则全扣，如果出现一个翻译错的单词，则扣1分。英语作文用有道字典的作文批改功能来批改。

等你的分数出来以后，就看一下你目标学校的对应年份的复试均分线。我把分数线分为复试分数线和复试均分线。复试分数线是指进入复试的最低分，复试均分线是指那些进入复试的人的总均分。

接下来就是判断自己是不是选择了正确的学校。如果要选择一个稳一点的学校，则你的数学加英语的分数大于等于目标学校的数学加英语的进入复试的均分，同时，四门总分要比复试均分高20分（如果没有考专业课和政治，则用均分代替）。以我当时的模拟分数为例，我的23年的数学在130左右，英语在80左右。数学加英语为130 + 80 = 210 大于海大的115 + 79（数学均分加英语均分）。同时，总分为130 + 80 + 60 + 104 （数学 + 英语 + 政治 + 专业课）= 374 大于海大总分均分的360，高出了14分。因此我选了这个学校。

如果要想平衡一些，则总分等于进入复试均分。如果你想去赌一下，则总分大于进入复试的最低分。不过我不推荐赌，很危险。

选择学校还要看它的复试内容，学校是不是歧视双非的学生，学校相关专业的排名情况，学校所在地的压分情况等等。这些直接在网上搜就可以了。

好好复习，打牢基础。在报名的时候发现，除了国内几个顶级大学没有把握可以上，其他大学报了就可以上的感觉真的很爽。＼(((￣(￣(￣▽￣)￣)￣)))／

政治

政治我听了徐涛的强化课，不过我感觉没啥用，正式开始背政治的时候，之前看的课也都忘了。但是徐涛讲的也是蛮好的，听课也没有压力。

政治的学习重点就是肖四肖八。肖八写选择，肖四写大题。肖四要背大题，每一个题都要背。如果没时间背，那就背前两套。大题也可以去找一下其他up主的整理视频，我跟着是大牙的整理文档。

复习时间安排

政治的学习方法：在12月份初的时候，肖八与肖四会出来，主要就是背肖八，肖四。我自己是自己写肖四，肖八，然后去b站上看 大牙考研 这个up主的相关的讲解视频。

肖八可以背一下马原部分的大题。不过主要还是背选择题，去看b站上的带背视频，那种视频会把一类相关的知识点都说出来。

英语

单词

英语从现在开始就要背单词，每天都背，把大纲的单词全背了。可以用手机上的背单词软件来背。如果有时间，可以再背背词组，词组也是蛮重要的，考试中会出现的，比如crop up miss out等。

大部分的单词都只需要留下印象，看到以后知道是什么意思，不用会写。只需要把作文中会用到的词能写出来就可以了。

我是从墨墨背单词上学的。背了以下几个单词书：2024硕士研究生英语（一）大纲词汇 2024恋恋不忘考研英语词组 2023墨墨考研深度记忆宝典 核心词

大三下学期一天背150个，大四上一天背300个。考前一个月就不背新的了，去背政治，然后复习旧单词。

长难句与语法

如果六级没过，那可以去看看，如果已经过了，那就不需要看了。如果你不放心也可以去看看。我看的是田静的，讲的蛮好，例句给的也多，不过基础好的就没必要看了。

阅读

我推荐去看 唐迟 的阅读课，我就是看了他的课以后，从每篇错2个降低到每篇错1个或没错。讲的蛮好的，直接去b站上看他的公开课就足够了，不用买付费课。

重点是学他的解题思路。如果英语好的同学，那用自己的方法写就可以了。如果是英语不好的同学（比如我），那么可以像我一样：先去听一下唐迟的解题方法与思路，然后完全放弃自己的方法，使用他的方法来写阅读（从阅读文章到写完题目）。多用几篇阅读练习一下就可以掌握，我个人认为效果很好。

完形填空

我看的是 宋逸轩 的课，感觉没啥用，关键还是以我的原则来写。

这两个题型的解题技巧总体方向是：从客观的角度来写，不要把自己代入作者。要牢记自己是一个旁观者，你不是作者。不要在写题的时候想自己觉得应该怎么做。

  • 阅读的原则是：不要自己加戏，紧跟着原文走。
  • 完形的原则是：不要把选项带入然后自己强行解释。应该把选项带入原文，然后去看上下文是否合理。

翻译与新题型

我看的也是 宋逸轩，讲的可以，可以去看看。
翻译主要听的是翻译思路，新题型听的是新题型的解题方法

作文

作文我推荐石雷鹏，讲的很好，牛犇啊。学习的重点直接听课就可以了。
作文的批改可以去有道字典里的作文改批功能来批。听说批改网也可以判分，不过我没用过。

我当时把10年以后的所有的作文都写了一遍。如果想要润色自己的语句，可以使用翻译软件翻译一下要写的句子，或者使用gpt来翻译自己的中文。比如：使用高级的英语单词与语法来将以下的内容从中文翻译成英文：[你要写的句子]。

复习时间的安排

推荐在7月开始学习除了作文以外的所有课，然后7月8月是强化阶段，这个时间就可以去写写阅读，完形，翻译啥的，练手。考英语一的就先写英语二的阅读，考英语二的就先写英语一的阅读。

到了9月份，就可以准备起作文了，学一个半月的作文，把石雷鹏的那几个模版作文都要背会。在10月中旬的时候，应该要把大作文和小作文的前三次课全学完，然后每天默一篇。一直持续到考研。期间可以自己去写写往年的真题的作文。我都写过，用石雷鹏的模版去写。练习作文的中文思路。

数学

时间段

基础：2022.12-2023.3（寒假）
强化：2023.3-2023.6（大三下）
刷题：2023.7-2023.12（暑假加大四上）

复习前期（冲刺前）

我学的是数学二，所以只对数学二提建议，学数学一的同学当个参考就可以了。

数学要准备一年，如果基础好的同学可以根据自己的进度来。我是在上一届考研结束了以后就开始学的。学数学的时候，一定要每天都写，不能断，要一直有手感。因为我之前是写完数学基础课以后就不写数学，只看专业课，从而导致我强化阶段像没学过一样。一般来说，高数2h，线代2h。学数学要一块时间的学，比如下午1点学到下午5点。数学要用1小时的时间进入心流状态，进入状态以后效率会很高。

高数部分，我跟的是 武忠祥 的全程。武神讲的很好，基础打的很牢。基础牢才可以写难题。不管是听哪个老师的课，一定要全程听他的基础课，强化课。并且，每个老师都有自己的练习题，所以在听课阶段，只需要跟紧老师就可以了，写这个老师的题目。每个老师都有自己的一个学习过程，他们出的题也是按照自己的学习过程来出的。我一开始的时候是听的武忠祥+660+1800。后来感觉不应该写1800。

线代部分，我跟的是李永乐的课，他的课很枯燥，但是知识点密度很大。如果各位有点受不了，可以去听听张宇的课，虽然我没听过他的课。听完以后，应该还是一头雾水，这个时候就可以去听一下没咋了的课。作为知识点的补充。

关于学习基础课与强化课的方式，给各位参考一下(●’◡’●)：我是学一章的课，然后就写一下书上对应的例题（一般这些题老师在课上都讲过），写完例题以后去写一下对应习题册的题目，最好把错题整理下来。然后整本书都写完以后，就重写一下之前写错的题目。这样的学习方式不会导致自己忘记的太快。核心就是反复多练。

复习后期（冲刺）

关于考试时写题的顺序，我是将题目的分值当成写题的时间限制，比如一个选择题是5分，那么我应该在5分钟之内写出来，如果超时了就一定要跳过。不过在考试时，我是先把会写的写了，一眼下去没思路的题目，我就直接跳过。这样我可以在很短的时间（1.5h）内将试卷上所有会写的，能确定的题目都写完。然后剩下的时间可以专门去啃那些不会写或者是没把握的题目。不过，我推荐的是写完以后先检查一遍试卷，验算一下，确保会写的题目没有算错的情况。我当时是花了半小时验算的。所以我用了最后1小时去啃那些硬骨头。这个方法不适合直接在考试中使用。推荐先用几套模拟卷自己感受一下后再在考试中用。考试的流程如下：写会写的题目 -> 验算已经写好的题目 -> 写不确定的题目 -> 写不会写的题目 -> 验算刚刚写的题目 -> 最后再验算所有的题目。考试的时候还要注意，要看清答题卡上的题号，不要填错位置了。

关于冲刺阶段的时间安排。我在整个11月，是一天一套模拟卷。同时还要写好相应的错题本。写模拟卷之前，可以先去网上看看其他人对你要写的那个模拟卷的评价，尽量写评价高的模拟卷。然后每写完一本模拟卷，要准备写下一本模拟卷之前，回顾一下之前写好的模拟卷的错题本，再重新写一下之前写错的题目（模拟卷上的题）。12月的时候。我是2天一套模拟卷，1天写，1天看之前写好的错题本（所有的错题）。其实如果你按我下面那种方式来复习错题本，错题本上的内容是不多的。1星期是基本可以复习完的。写模拟卷的时候，可以留1到2套用于在考试前预热。我当时留的是张宇4套卷的最后两套卷。临考前一天一套。然后再复习一下就可以了。如果你的选择题不太行，那么也可以在12月份写一下各家模拟卷选择题部分。具体怎么样复习就要看各位的随机应变了。复习的时候要多想：有没有什么更好的方式来复习，如果有，那应该是怎么复习，相比现在的复习方式有什么改进的地方。

高数笔记

这个是我的笔记，还有错题本。笔记全都记在书上，后期复习的时候直接看书。把书上会写的例题全都划了，只留下不会写的例题。错题本也是，把写对的题目划了，只留下不会写的题目。在错题本上，黑笔写的是答案，红笔写的是从别人的视频那看来的解题思路。

忠告

  • 数学不能怕犯错，在后期写卷子的时候也不能因为怕考不高而不写，写卷子的目的是筛选自己还有哪里不会的，而不是看自己最后会考多少分，不要关注分数，关注自己犯错的点
  • 要有错题本，每一本书写完以后，都要对着错题本再写一次。错题本上可以记一下什么书的第几页的第几题，然后写一下解答过程。我常用的题目定位的格式是：[习题集]-[页号]-[题号]或[习题卷]-[卷号]-[题号]，比如我上文给出的图片，超越卷1 1的意思就是超越卷 的 第1套卷 的 第1题 写错了。错题本可以用活页本，一个页的题目全都会写以后直接把这一页拆下来即可。关于错题本的复习顺序：我一般是写完一本习题册，就去回顾一下错题本。主要的原因如下：考研的题目种类就那么多，很少有习题册有新的内容。所以如果你写好A册，接下来要写b册。如果你没有回顾A的错题本，那么在B中见到了类似的题目，你还是有很大的可能会出错。如果你回顾了一下。那么你写b的时候，就会有很大的概率是写对的。这样可以减少错题本上相似题目的冗余，还可以减少复习错题本所花费的时间，你还能有成就感，三赢！
  • 要记住，考研的数学一定是会有创新题的，就是那些你以前没有写过的题目，没见过的出题方法。所以，想要冲高分的同学一定要写一下创新卷。比如张宇的4套卷，经过4套卷的洗礼，你在写真题卷的时候就会比较镇定。
  • 我在上面提到的所有的题目，都可以在b站上找到相应的讲解视频，我个人推荐的是去b站上找对应的讲解视频，去看看别人的解题思路是什么样的，学解题思路。我这里推荐一下没咋了，我看的是他880，李林6+4讲解，讲的很好。他的线代课也一级棒，不过推荐有基础了以后再听，先听各大名师的课。
  • 写到计算题的时候，一定要算到最后，如果算错了，那么一定要把这个题目加入错题集，下次再做一下，计算是你有基本功，一定要练好了，不可以存在有思路就不写的情况。我自己在算数学模拟卷的考试分数时，我选择将数学的分数划分为 总分，失误分，危险分。总分就是当前这个卷子最后的分数，失误分为因计算失误，写错与选错所失去的分数。危险分为本次写对了，但是不知道这个题目的解题过程的分数。
  • 如果在写一个题目或听课的时候感觉到吃力了，说明这个内容是你的薄弱点，要着重突破。只有不断跳出自己的舒适区，能力才可以提升，写出来1道原本不会题要比写出来10道会写的题强。这个过程是很缓慢，同时也是很累的，非常容易产生挫败感。因此需要做好心理准备，尤其是在基础阶段与强化阶段。
  • 一定要以自己的实际情况为主，我上面的那些都只能做一个参考。

专业课408

我全程听着王道的基础课加强化课，看着王道书，已经足够应对408了，视频就看b站上的免费视频就可以了。强化课的视频要通过各位用点手段来搞到了。我在7月份才决定学的408，所以我在7月份才学的计组和计网的基础课，因此你们也是有时间去学的。

408的内容又多又杂，推荐各位在第一遍学的时候，写书后面的小题的时候，先写到草稿纸上，然后书上不要有痕迹，然后在8-9月第二轮复习的时候再写到书上。这样相当于写了两遍，记的深。

王道书后面的小题我写了蛮多遍的，再配合着王道的模拟卷的小题就已经足够应付考研了。可以把写错的小题的解析写到一个小笔记本上，然后日后复习的时候直接对着笔记本复习就可以了，我前段时间就这样复习的，一个小时就可以复习完模拟卷和真题卷的题目，速度嘎嘎快。

我在10月份写的真题卷，不过我推荐可以早点写，然后在12月的时候再写一遍真题卷的大题。王道的强化课是比较推荐的，讲大题的解题方法与知识点的综合，听完会有茅塞顿开的感觉。

王道在12月中旬的时候会有一个冲刺课，可以去报个名听一下，咸鱼会把最后几天的重点都勾出来，然后剩下的几天如果没东西复习了，就可以跟着他的重点再复习一次。

关于王道书的书后大题，是针对性的写的，比如：408的试卷在考到操作系统的部分时，经常会考到二级页表相关的内容，那我就会把书上二级页表相关的大题全写一遍。其他的大题我就基本没有写。

要不要去辅导机构

没必要。如果你有良好的自制力，那么你看网课就已经足够了。如果你没有自制力，那么报了班也没啥用，更何况，那些机构老师的水平还不如考研名师。

要留校吗

看自己的情况，如果在家的效率没有在学校的效率高，那可以考虑一下留校。如果在家学习的效率和在学校的效率一样，则可以不考虑。

演讲稿

各位老师们、同学们，大家晚上好，我是来自xxxxx。很荣幸在这里分享我的考研经验。
入校之际，我就给大学生涯做了简单的规划，在大一与大二时，以拓展项目经验，竞赛经验为主。跟着老师们参加各种比赛，各类项目，如蓝桥杯比赛等。在大三下学期，我正式开始考研的复习。我认为跟着老师参加比赛或者做一些科研，在考研时很有优势，尤其是在复试的时候很容易被导师认可，最后实现逆袭。
我将整个考研阶段分成了3部分：基础阶段，强化阶段与刷题阶段。基础阶段在4月份结束，强化阶段在7月份结束，剩下的时间就是不断的刷题。在一天的时间安排上，我给数学4小时，专业课3小时，英语阅读1小时，其他零碎的时间用来背英语单词，比如在食堂等饭，晚上临睡前等等，这些都是我的个人情况定制的计划，要考研的学弟学妹们可以根据自己的实际情况制定考研计划。
无论是高数还是线代，一定要把基础打牢，课堂上做好笔记，书本上的每一道例题和习题都不放过。复习数学时，我坚持每天都写，不间断，保持做题手感。在时间安排上，我每天用一整块的时间学，给高数2小时，线代2小时，进入状态很重要。复习数学时一定要记住，没有什么新的知识点，题目的解题方法都在强化阶段讲过。因此，一定要及时总结各种题型的解题方法。再说说我复习数学另一个心得，那就是错题本。我用活页本当作错题本，一整页的题目全都会写以后直接把这一页拆下来即可，这样错题本会越写越薄。一般来说，我在错题本上用黑笔写写错的题目与题目的解析；用红笔写解题思路和没关联起来的知识点。但是，如果把题目全都抄下来又有些浪费时间，所以我一般在错题本上标注题目所在的书籍和页码，然后写一下解答过程。我常用以下3个关键字来定位一个题目：[习题集名字]-[页号]-[题号]或[习题卷名字]-[卷号]-[题号]。到后期阶段，哪怕模拟试卷分数很高，也不要觉得自己没问题了，还是要保持良好的心态，多刷题，多分析自己出错的原因。
学弟学妹们，如果在写一个题目或听课的时候感觉到吃力了，说明这个内容是你的薄弱点，要着重突破。只有不断跳出自己的舒适区，能力才可以提升，写出来1道原本不会题要比写出来10道会写的题强。这个过程是很缓慢，同时也是很累的，非常容易产生挫败感。因此需要做好心理准备，尤其是在基础阶段与强化阶段。
学弟学妹们，专业课408的学习方法我想跟大家也分享一下。408本身其实考的不算深，但是它考的内容很广，所以在平时一定要多看书。学408就要像学数学一样，每天都看，比如错题，书上的知识点等。等把所有的题目都写完以后，根据你之前刷过的题目，自己画画思维导图。画的过程就是重新整理考试重点的过程。我的专业课的错题本是直接写题目中的知识点与题号，定位题目的方法与刚刚说定位数学题目的方法一致。在复习的时候直接看这个题目考的知识点，如果对这个知识点的考法出现了遗忘，那么就可以根据题目来回顾。
在备考期间，一定要注意自己的休息，保证每天7小时以上的睡眠时间。同时，多跑步运动，有一个健康的身体才可以好好学习。
不积跬步无以至千里，希望大家都能打牢基础，成功上岸，我的分享到此结束，谢谢大家！
